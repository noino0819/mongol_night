## 1. 스토리 골격 — "사흘, 세 조각"

여행 첫날 밤 고비에 떨어진 "고비의 별"이 **세 조각**으로 갈라졌다.
하루에 한 조각씩, 사흘의 여정 = 3챕터. 텡그리(별 늑대 정령)가 GM.

| 챕터 | 무대 | 조각 획득처 | 크로스오버 미니게임 | 몽골 소재 |
|---|---|---|---|---|
| 프롤로그 | 게르 캠프의 밤 | — (스탯 배분) | — | 게르, 별자리 |
| 1장 첫째 날 "초원의 흔적" | 초원 → 협곡 | 협곡 바위틈 (판정) | 샤가이 대결 → 실제 가위바위보 | 독수리 사냥꾼, 아이락, 오보, 샤가이 |
| 2장 둘째 날 "모래폭풍의 밤" | 고비 사막 | 폭풍의 눈 회오리 (판정) | 텔레파시 → 전원 동시에 같은 별 지목 | 모래폭풍, 우물, 카라반, 마두금 |
| 3장 셋째 날 "늑대의 시험" | 크레이터 | 늑대왕 시험2 보상 (판정) | 인디언포커풍 담력 → 실제 눈싸움 | 늑대, 늑대왕, 별 본체 |

**텡그리 캐릭터 아크**: 츤데레 의뢰인(프롤로그) → 잔소리 동행자(1·2장) → 3장에서 이상하게 침묵 → 최종전에서 몸을 던져 그림자를 붙듦 → 정체 공개: **별을 지키는 수호 늑대 정령** (늑대왕 무리와 동족). 연출 디테일은 텡그리 연출 턴에서.

**최종보스 "별의 그림자"**: 별이 떨어지며 생긴 검은 모래 회오리. 늑대왕은 적이 아니라 시험관 — "별을 지키는 건 우리가 아니다"로 복선.

## 2. 상태 변수 & 스키마 확장

```js
vars: { 용기:2, 지혜:2, 인심:2,  // 0~5 클램프
        sheep:5,                  // 0 이상 클램프
        star:0,                   // 별조각 0~3
        charm:0,                  // 폭풍 부적 (0/1)
        fails:0 }                 // 주요 판정 실패 수 — 방랑 엔딩 연출 분기용
```

기획서 4-3 노드 스키마에 **판정 로직 턴에서 구현할 확장 3종**:

1. **`auto` 노드** — 조건 순차 평가 무선택 분기. 마지막 항목은 무조건 goto.
   `auto: [{if:"sheep>=10", goto:"c3_29"}, {goto:"c3_30"}]`
   조건식은 상태 변수명으로 구성된 JS 표현식 (`용기+지혜+인심>=9` 등).
2. **choice 레벨 `set`** — 스탯 배분 선택지용 (`{label:"용기+1", set:{용기:1}, goto:...}`). 노드 레벨 `set`(진입 시 적용)과 별개로, 선택 확정 시 적용. `cost`와 동일 타이밍.
3. **`fails` 카운터** — 조각 상실(c1_24, c2_28)·시험 실패(c3_14/18/22) 노드의 `set:{fails:1}`. 엔딩 조건에는 안 쓰고 방랑 엔딩 텍스트 강도 분기에만 사용 (fails≥4면 "텡그리가 혀를 차다 못해 웃는다" 톤).

기타 시맨틱: `set`/`cost`는 델타. `hideIf`는 조건식 참이면 선택지 숨김 (`"sheep<2"`, `"charm<1"`). check 실패는 즉사 없음 — 전부 우회/손실 분기 (게임오버는 c3_26 단 1곳).

## 3. 판정 밸런스 (스탯 + d6 ≥ dc)

| dc | 스탯 2 | 스탯 3 | 스탯 4 | 용도 |
|---|---|---|---|---|
| 5 | 67% | 83% | 100% | 워밍업 (c1_02 추격, c2_09 우물) |
| 6 | 50% | 67% | 83% | 표준 (조각 획득 1차, 시험 1차) |
| 7 | 33% | 50% | 67% | 어려움 (2차 만회, 최종전, 담판) |

dc 8은 미사용 (스탯 2에서 17%는 파티게임에 좌절감 과다). 실패 시 반드시 ①양 지불 우회 ②2차 판정 ③손실 후 진행 중 하나가 열린다. **양 지불 우회는 확정 성공** — 자원으로 운을 사는 구조가 양 경제에 긴장을 만든다.

## 4. 챕터 분기 구조

### 프롤로그 (p_01~07)
직선 7노드. p_05에서 스탯 배분 다수결 (+1, 스탯합 7 시작).

### 1장 (c1_01~25) — 2루트 → 오보 합류 → 협곡 클라이맥스
```
c1_01 ─ 독수리A: 02(용기5) → 03/04 → 05(훈련 용기6→06 용기+1 / 07) → 08 ─┐
      └ 게르B:   09 → 10(샤가이? 11:가위바위보 → 12 양+1 / 13 인심+1        ├→ 17 오보
                        / 사양 14(인심6) ) → 15 양+2 / 16 → ──────────────┘
c1_17 오보 ─ 양1 바침 → 18 스탯+1 선택 / 돌만 → 19 "쩨쩨하군"
→ 20 협곡 → 21 조각 판정(용기6/지혜6) ─ 성공 → 22 ⭐star+1
                                      └ 실패 → 23 (밧줄 지혜7 / 🐑여우 확정) ─ 실패 → 24 상실 fails+1
→ 25 밤 캠프 → 2장
```
루트 성격: A 독수리 = 스탯(용기) 루트, B 게르 = 양(+1~+3) 루트. **목동 노리면 B 강제** — 전략성.

### 2장 (c2_01~31) — 2루트 → 카라반 → 게르의 밤 → 모래폭풍
```
c2_03 ─ 직행A: 04(용기6) → 05 용기+1 / 06 → 07 ─┐
      └ 우물B: 08 → 09(지혜5) → 10 지혜+1 → 11 ──┴→ 12 카라반
c2_12 카라반 ─ 부적 🐑2 → 13 charm / 흥정(인심7) → 14 charm 🐑1 / 길만 → 15
c2_15 상인 부탁(지혜6) → 16 양+2 / 17 → 18 게르 재회
→ 19 텔레파시 → 20 양+1 인심+1 / 21 → 22 자정
→ 23 폭풍 진입(용기7/지혜6/🐑1 길잡이 확정) ─ 실패 → 26 양-1 (재돌진 용기6 / 대피)
→ 27 폭풍의 눈(인심6 인간탑 / charm 확정 / 용기7 점프) ─ 성공 → 29 ⭐star+1
                                                        └ 실패 → 28 매몰 fails+1
→ 30 귀환 → 31 → 3장
```
부적(charm)은 **2장 폭풍의 눈과 3장 최종전 양쪽의 보험** — 양 2마리의 가치가 있다.

### 3장 (c3_01~33) — 담판 → 3시험 → 최종전 → 엔딩 게이트
```
c3_01[체크포인트] → 02 늑대왕 → 03 담판(용기6/인심6/🐑2 예물 확정)
  실패 → 05 새끼 늑대 (구출 → 06 인심+1 / 잠입 07: 지혜7 → 09 개그 / 10 들킴) → 전부 08 시험행
시험1 담력: 11 눈싸움[크로스오버] → 패배 시 13(용기6 만회) → 실패 14 fails+1
시험2 눈:   15(지혜6) → 성공 16 ⭐star+1(셋째 조각) / 실패 17(지혜7 재도전) → 18 상실 fails+1
시험3 마음: 19 소원 선택(3지선다, 전부 20행) → 20(인심6) → 21 / 22 fails+1
→ 23 → 24 그림자 각성 → 25 최종전(용기7 / charm 확정) ─ 실패 → 26 손잡기(인심7) ─ 실패 → 💀 end_go
→ 27 텡그리 헌신 → 28 [게이트1] sheep≥10 → 29 목동 선택 → end_sheep or ↓
→ 30 [게이트2] star3+스탯합9 → 33 → ⭐end_star
                star≥2 → 31 담판(지혜7) → 성공 33 / 실패 32
                else   → 32 → 🌪️end_wander
```

## 5. 엔딩 매트릭스

| 엔딩 | 조건 (c3_28→30 게이트) | 도달 전략 |
|---|---|---|
| ⭐ 진 | star 3 && 스탯합 ≥9 (또는 star≥2로 담판 지혜7 성공) | 판정 성공 + 스탯 성장 선택 축적 |
| 🐑 목동 | sheep ≥10 && "남는다" 선택 | 게르 루트·샤가이 승·카라반 부탁·무지출 (최대 12마리 가능) |
| 🌪️ 방랑 | star ≤1 또는 담판 실패 | 조각 2개 이상 상실 |
| 💀 게임오버 | c3_25·26 연속 실패 (유일한 사망 지점) | 체크포인트 c3_01 재시작 (스냅샷 복원) |

- 스탯합 9 = 시작 7 + 성장 2회. 성장 기회는 런당 최대 6회(프롤로그·1장 루트·오보·2장 루트·텔레파시·새끼 늑대) — 달성 가능하되 자동은 아님.
- 진엔딩의 star 3 && 합 <9 케이스는 게이트2에서 star≥2 분기로 떨어져 **담판 재기회** — "별은 모았지만 그릇이 부족" 서사로 처리.
- 목동은 별 3개를 다 모으고도 양을 고를 수 있는 **개그 배신 엔딩** (c3_29에서 선택).

## 6. 양 경제 (시작 5)

| 수입 | 지출 (전부 선택적) |
|---|---|
| 샤가이 승리 +1 (c1_12) | 오보 봉헌 -1 → 스탯+1 |
| 가족 선물 +2 (c1_15) | 협곡 여우 -1 → 조각 확정 |
| 카라반 사례 +2 (c2_16) | 부적 -2 (흥정 성공 시 -1) |
| 소녀 선물 +1 (c2_20) | 폭풍 길잡이 -1 → 안전 진입 |
| | 폭풍 휩쓸림 -1 (c2_26, 벌칙) |
| | 늑대왕 예물 -2 → 담판 스킵 |

최대 +6 / 최대 -8. "양을 쓰면 안전, 아끼면 목동" — 매 지출 선택이 목동 엔딩과의 트레이드오프.

```

✅ 🌪️ 방랑엔딩 도달 (star 0, fails 5)
```

**판정 로직 턴 재검증 (2026-07-21)**: 랜덤 분포 진 46.7% / 방랑 37.9% / 게임오버 15.2% / 목동 0.2%로 갱신 — 설계 턴 시뮬레이터가 choice 레벨 `set`(스탯 배분·오보 소원)을 미적용해 스탯 성장을 과소평가했던 것. 엔진과 동일 로직으로 재측정한 위 수치가 기준. 파티게임 특성상 랜덤 기준 진엔딩 ~47%는 허용 범위 — 최종 난이도 튜닝은 저작·QA 턴에서.
목동 0.1%는 의도된 값 (전략적 사재기 없이는 안 나오는 히든성 엔딩). 게임오버 23%는 무작위 선택 기준이며 실제 플레이(양 우회·부적 보험 활용)에서는 크게 낮아짐 — 저작 턴에서 c3_25 진입 전 양/부적 힌트 대사로 보정.
